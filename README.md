# # HormigasAIS - Demo Sound

Este repositorio contiene un **mini ejemplo de sonido creado con Furnace Tracker**.

## Contenido

- `sounds/demo.fur` → Archivo original de Furnace (proyecto editable)
- `sounds/demo.wav` → Exportación lista para reproducir en la web
- `index.html` → Página web simple para reproducir el sonido

## Uso

1. Abrir `index.html` en cualquier navegador.
2. Hacer click en el botón de reproducción.
3. Disfrutar del sonido.

## Contribución

- Puedes agregar nuevos sonidos exportados desde Furnace dentro de `sounds/`.
- Mantener un `.fur` por cada WAV es recomendable para control de versiones.
